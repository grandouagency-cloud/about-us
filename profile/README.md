## Who We Are

**GRANDOU Agency** is a Haitian digital agency founded to transform the online presence of businesses in Haiti and the diaspora. We craft identities that leave a mark, web experiences that engage, and growth systems that last.

We don't just build websites — we build **digital assets**.

```
  Identity              Web Experiences          Systems

  Branding              SPA / Landing            Automation

  UI / UX               Client Portals           SEO / Analytics
  Design System         Full Redesigns           Email Marketing
```
